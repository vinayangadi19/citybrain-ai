# 🚦 CityBrain AI – Smart Traffic Intelligence Platform

A full-stack, AI-powered smart city platform designed to analyze, predict, and reduce road accidents using Machine Learning, Geospatial Intelligence, and Real-Time Analytics.

---

## 🚀 Key Features

- Real-time Accident Risk Prediction
- Blackspot Detection & Heatmaps
- Smart Infrastructure Recommendations
- Sentiment Analysis
- Insurance Prediction
- Explainable AI

---

## ⚙️ Run Project

1. Install dependencies:
pip install -r requirements.txt

2. Generate data:
python data/generate_mock_data.py

3. Train model:
python src/model.py

4. Run app:
streamlit run app.py
